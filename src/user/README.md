# 用户配置目录

## 目录说明

此目录用于存储用户配置文件，实现多设备配置同步。

## 文件命名规则

- 文件名：`{user}.json`
- user 参数：只允许字母、数字、下划线、中划线

## 文件结构

每个用户的配置文件支持多个模式：

```json
{
  "configs": {
    "default": { /* 默认模式配置 */ },
    "team1": { /* team1 模式配置 */ },
    "class1": { /* class1 模式配置 */ }
  },
  "histories": {
    "default": [ /* 默认模式历史 */ ],
    "team1": [ /* team1 模式历史 */ ]
  },
  "createdAt": "2026-04-09T00:00:00.000Z",
  "updatedAt": "2026-04-09T01:00:00.000Z"
}
```

## 权限要求

- 目录权限：可写（777）
- 文件权限：可写（666）

## 注意事项

- 此目录需要部署 API 才能生效
- Node.js API：`src/api/api.js`
- 定期清理旧配置文件，避免占用过多空间

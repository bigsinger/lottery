# 抽奖网页程序 - 使用说明

## 快速开始

### 1. 本地使用

直接双击打开 `src/index.html` 文件即可开始使用。

### 2. 部署到服务器

将 `src` 目录下的所有文件上传到虚拟主机，访问对应 URL 即可使用。

---

## 应用场景

本程序是一款灵活的转盘抽奖工具，支持两种主要应用场景：

### 场景一：抽奖品

适用于各类活动、促销、年会等场景，设置奖品列表进行抽奖。

**示例配置**：
- 一等奖：iPhone 15
- 二等奖：京东礼品卡
- 三等奖：精美笔记本
- 谢谢参与

### 场景二：抽人

适用于公司、班级、团队等场景，设置人员名单进行随机抽取。

**典型应用**：
- 公司年会抽奖员工
- 班级随机点名回答问题
- 团队随机分配任务
- 会议随机选择发言人

**使用方法**：
1. 点击「导入」按钮，批量粘贴人员名单
2. 每行一个姓名，支持多人快速导入
3. 设置权重控制被抽中的概率
4. 设置最大中奖次数，避免重复抽中同一人
5. 点击「重置」按钮可开始新一轮抽取

**权重设置技巧**：
- 权重 0：该人员不会被抽中（如领导、已获奖者）
- 权重 100：该人员必定被抽中（如指定某人）
- 权重 10-50：正常参与随机抽取

## 基本功能

### 抽奖操作

1. 点击页面中央的「开始抽奖」按钮
2. 转盘开始旋转动画
3. 等待转盘停止，显示中奖结果
4. 结果弹窗会在 3 秒后自动关闭

### 转盘显示

- 文字在上、图标在下，文字显示区域更大
- 文字过长时会自动截断显示省略号
- 手机端已优化渲染，确保所有选项正确显示

## 配置设置

### 打开设置面板

点击右上角的「⚙」按钮打开设置面板。

### 修改标题

在设置面板中修改「抽奖标题」，保存后生效。

### 管理选项

#### 添加选项

1. 点击「+ 添加奖品」按钮
2. 输入选项名称
3. 选择图标和颜色
4. 设置中奖权重（0-100）
5. 设置最大中奖次数
6. 点击设置面板底部的「保存」按钮

#### 编辑选项

1. 在选项列表中点击「编辑」按钮
2. 页面自动滚动到编辑区域
3. 修改选项属性
4. 点击设置面板底部的「保存」按钮

#### 删除选项

1. 在选项列表中点击「删除」按钮
2. 确认删除操作
3. 注意：至少需要保留一个选项

#### 清空选项

点击「清空」按钮可一键清空所有选项。

#### 批量导入

点击「导入」按钮，在弹窗中粘贴选项名称：

- 每行一个名称
- 名称不超过10字
- 适用于抽人场景快速导入人员名单

## 选项属性说明

### 名称

选项的显示名称（奖品名或人名），会在转盘上显示。

### 图标

预制的 Emoji 图标，让转盘更加美观。如果不选择图标，会显示选项名称文字。

可选图标：
- 🎁 礼物
- 🏆 奖杯
- 💰 红包
- 🎫 门票
- 🎮 游戏
- 📱 手机
- 🍰 蛋糕
- ✨ 星星
- 🎈 气球
- 🌹 玫瑰
- 💎 钻石
- 🎯 目标

### 颜色

转盘扇形的背景颜色。

可选颜色：
- 🔴 #e74c3c 红色
- 🟠 #e67e22 橙色
- 🟡 #f1c40f 黄色
- 🟢 #2ecc71 绿色
- 🔵 #3498db 蓝色
- 🟣 #9b59b6 紫色
- 🟤 #1abc9c 青色
- ⚫ #34495e 深灰

### 权重（0-100百分比）

中奖概率的百分比，数值越大中奖概率越高。

**权重说明**：
- 权重 0：该选项永远不会被抽中
- 权重 100：该选项必定被抽中（100%概率）
- 权重 10-50：正常参与随机抽取

**示例**：
- 选项A 权重 10
- 选项B 权重 20
- 选项C 权重 50

选项C 的中奖概率最高，选项A 的中奖概率最低。

### 最大中奖次数

该选项最多可被抽中多少次（默认为1）。

- 达到最大中奖次数后，该选项不再参与抽奖
- 点击「重置」按钮可清零所有中奖次数，开始新一轮

## 多模式使用

### 什么是多模式？

通过 URL 参数 `?mode=xxx` 可以切换不同的抽奖模式，每个模式拥有独立的配置存储。

### 使用方法

在 URL 后添加 `?mode=xxx` 参数：

```
index.html?mode=team1
index.html?mode=team2
index.html?mode=class1
index.html?mode=company
```

### 应用场景

- **不同团队**：每个团队使用独立模式
- **不同场景**：奖励抽奖、惩罚抽奖、活动抽奖
- **不同人群**：孩子奖励、学生奖励、员工抽奖
- **抽人场景**：不同班级、不同部门使用不同模式

## 多设备同步

### 什么是多设备同步？

通过 URL 参数 `?user=xxx` 可以实现配置跨设备同步，同一用户在不同设备上配置自动同步。

### 使用方法

在 URL 后添加 `?user=xxx` 参数：

```
https://your-domain.com/lottery/index.html?user=张三
https://your-domain.com/lottery/index.html?user=李四&mode=team1
```

### 使用要求

- 需要部署到支持 PHP 的虚拟主机
- 确保 `user/` 目录有写入权限（chmod 755）
- GitHub Pages / Vercel / Netlify 不支持 PHP，无法使用此功能

### 配置存储

- 有 `user` 参数：配置保存到服务器 `user/{username}.json`
- 无 `user` 参数：配置保存到本地 localStorage

---

## 数据存储

### 存储方式

**本地模式**：
- 所有配置保存在浏览器的 localStorage 中
- 配置数据：`lottery_config_{mode}`
- 抽奖历史：`lottery_history_{mode}`

**服务端模式**：
- 配置保存到服务器 `user/{username}.json`
- 跨设备自动同步

### 注意事项

- 本地模式：清除浏览器数据会丢失配置，不同浏览器之间配置不共享
- 服务端模式：建议定期备份重要配置

## 常见问题

### Q: 配置丢失了怎么办？

A: 点击设置面板中的「重置」按钮，恢复默认配置。

### Q: 转盘不显示怎么办？

A: 请检查浏览器是否支持 Canvas，建议使用现代浏览器（Chrome、Firefox、Safari、Edge）。

### Q: 抽奖结果不公平怎么办？

A: 检查选项权重设置，调整权重值可以改变中奖概率。

### Q: 如何避免重复抽中同一人？

A: 设置最大中奖次数为1，抽中一次后该人不再参与抽奖。

### Q: 所有选项都抽完了怎么办？

A: 点击「重置」按钮清零中奖次数，开始新一轮抽取。

### Q: 如何快速导入人员名单？

A: 点击「导入」按钮，批量粘贴姓名（每行一个），快速导入。

### Q: 如何让某人不被抽中？

A: 将该人的权重设置为0，权重0表示永不中奖。

### Q: 如何指定某人被抽中？

A: 将该人的权重设置为100，权重100表示必中。

### Q: 手机端转盘显示异常怎么办？

A: 本程序已优化手机端渲染，如仍有问题请刷新页面或使用其他浏览器。

### Q: 编辑后如何保存？

A: 点击编辑按钮后，页面会自动滚动到编辑区域，修改完成后点击设置面板底部的「保存」按钮。

### Q: 如何在不同设备上使用相同配置？

A: 使用支持 PHP 的虚拟主机部署，在 URL 中添加 `?user=你的名字` 参数。

# 抽奖网页程序 - 设计文档

## 0. 产品定位

### 0.1 核心功能

本程序是一款灵活的转盘抽奖工具，支持两种主要应用场景：

**场景一：抽奖品**
- 适用于各类活动、促销、年会等场景
- 设置奖品列表进行随机抽奖
- 支持权重控制中奖概率

**场景二：抽人**
- 适用于公司、班级、团队等场景
- 设置人员名单进行随机抽取
- 典型应用：年会抽奖员工、班级随机点名、团队随机分配任务、会议随机选择发言人
- 支持批量导入人员名单（每行一个姓名）
- 支持设置最大中奖次数，避免重复抽中同一人
- 支持权重设置（权重0=不参与抽取，权重100=必定抽中）

### 0.2 功能特性

- 纯前端实现，无需后端服务
- 支持多模式独立配置
- 支持权重概率控制（0-100百分比）
- 支持批量导入名单
- 支持中奖次数限制
- 支持中奖次数可视化显示
- 响应式设计，支持移动端（已优化手机端渲染）

## 6. 关键算法

### 6.1 权重抽奖算法（百分比制）

```javascript
function weightedRandom(prizes) {
  // 过滤已达到最大中奖次数的选项
  const availablePrizes = prizes.filter(p => p.winCount < p.maxWins);
  
  if (availablePrizes.length === 0) {
    return null; // 所有选项都已抽完
  }
  
  // 计算总权重
  const totalWeight = availablePrizes.reduce(
    (sum, p) => sum + (p.weight || 10), 0
  );
  
  // 生成随机数
  let random = Math.random() * totalWeight;
  
  // 找到中奖选项
  for (let i = 0; i < availablePrizes.length; i++) {
    random -= availablePrizes[i].weight || 10;
    if (random <= 0) {
      return availablePrizes[i];
    }
  }
  
  return availablePrizes[0];
}
```

### 6.2 转盘角度计算

```javascript
function calculateRotation(prizeIndex, totalPrizes) {
  // 每个选项占的角度
  const anglePerPrize = 360 / totalPrizes;
  
  // 目标角度（从顶部开始）
  const targetAngle = prizeIndex * anglePerPrize + anglePerPrize / 2;
  
  // 多转几圈 + 目标角度
  const totalRotation = 360 * 5 + (360 - targetAngle);
  
  return totalRotation;
}
```

### 6.3 Canvas 绘制转盘

```javascript
function drawTurntable(ctx, prizes, centerX, centerY, radius) {
  const anglePerPrize = (2 * Math.PI) / prizes.length;
  
  prizes.forEach((prize, index) => {
    ctx.save();
    ctx.beginPath();
    ctx.moveTo(centerX, centerY);
    ctx.arc(centerX, centerY, radius, 
            index * anglePerPrize - Math.PI / 2, 
            (index + 1) * anglePerPrize - Math.PI / 2);
    ctx.closePath();
    
    // 填充颜色
    ctx.fillStyle = prize.color || getDefaultColor(index);
    ctx.fill();
    
    // 绘制文字或图标
    ctx.fillStyle = '#fff';
    ctx.font = '14px Arial';
    ctx.textAlign = 'center';
    
    const text = prize.icon || prize.text;
    ctx.fillText(text, 
                 centerX + Math.cos(angle) * radius * 0.7,
                 centerY + Math.sin(angle) * radius * 0.7);
    
    ctx.restore();
  });
}
```

## 9. 性能优化

### 9.1 加载优化

- CSS/JS 内联或合并，减少 HTTP 请求
- 图片使用 Emoji 代替，减少资源加载
- 使用 CSS Transform 实现动画，启用 GPU 加速

### 9.2 运行优化

- Canvas 绘制缓存，避免重复绘制
- 事件防抖，避免频繁触发
- 配置变更时才重绘转盘

### 9.3 手机端优化

- 动态计算 Canvas 尺寸，适配不同屏幕
- 添加最小尺寸约束，避免渲染失败
- 改进 DPR 处理，确保清晰度

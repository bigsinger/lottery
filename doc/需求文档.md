# 抽奖网页程序 - 需求文档

## 1. 项目概述

开发一个基于 HTML/CSS/JS 的转盘抽奖网页程序，支持个性化配置、本地存储、多模式切换，可部署在简单的虚拟主机上。

## 2. 技术约束

### 2.1 技术栈限制
- ✅ HTML5 + CSS3 + JavaScript（原生）
- ✅ 可使用 Vue.js（通过 CDN 引入）
- ❌ 不允许使用其他复杂框架
- ✅ 使用相对路径，确保本地和服务器效果一致

### 2.2 部署要求
- 纯静态文件，无需后端服务
- 上传 src 目录即可立即生效
- 本地双击 HTML 即可预览，所见即所得

### 2.3 兼容性
- 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- 支持移动端响应式布局

## 3. 功能需求

### 3.1 核心功能 - 转盘抽奖

| 功能项 | 描述 | 优先级 |
|--------|------|--------|
| 转盘绘制 | 使用 Canvas 绘制转盘，显示奖品选项 | P0 |
| 抽奖动画 | 点击按钮后转盘旋转动画 | P0 |
| 中奖提示 | 抽奖结束后显示中奖结果 | P0 |
| 权重概率 | 支持为每个选项设置中奖权重 | P1 |

### 3.2 配置功能

| 功能项 | 描述 | 优先级 |
|--------|------|--------|
| 设置按钮 | 右上角设置按钮，点击打开配置面板 | P0 |
| 标题修改 | 允许用户自定义抽奖标题 | P0 |
| 奖品配置 | 添加/编辑/删除奖品选项 | P0 |
| 图标选择 | 为奖品选择预制图标或使用颜色背景 | P1 |
| 概率配置 | 为每个奖品设置中奖权重 | P1 |
| 黑白名单 | 配置中奖名单过滤规则 | P1 |

### 3.3 数据持久化

| 功能项 | 描述 | 优先级 |
|--------|------|--------|
| 本地存储 | 使用 localStorage 存储配置 | P0 |
| 永久有效 | 配置永久有效，不过期 | P0 |
| 模式隔离 | 不同 mode 参数使用独立配置 | P0 |

### 3.4 多模式支持

| 功能项 | 描述 | 优先级 |
|--------|------|--------|
| URL 参数 | 通过 ?mode=xxx 区分不同抽奖模式 | P0 |
| 独立配置 | 每个 mode 拥有独立的配置存储 | P0 |
| 默认模式 | 无 mode 参数时使用默认配置 | P0 |

### 3.5 帮助功能

| 功能项 | 描述 | 优先级 |
|--------|------|--------|
| 问号按钮 | 右上角问号按钮，点击查看使用说明 | P0 |
| 使用说明 | 清晰的功能说明和操作指南 | P0 |

## 4. 界面需求

### 4.1 主界面布局

```
┌─────────────────────────────────────┐
│  [?] [⚙]           [标题]           │
├─────────────────────────────────────┤
│                                     │
│           ┌─────────┐               │
│           │         │               │
│           │  转盘   │               │
│           │         │               │
│           └────┬────┘               │
│                │                    │
│           [抽奖按钮]                │
│                                     │
└─────────────────────────────────────┘
```

### 4.2 设置面板

```
┌─────────────────────────────────────┐
│  设置                           [×] │
├─────────────────────────────────────┤
│  抽奖标题: [________________]       │
│                                     │
│  奖品列表:                          │
│  ┌─────────────────────────────┐   │
│  │ 奖品1 [编辑] [删除]         │   │
│  │ 奖品2 [编辑] [删除]         │   │
│  │ [+ 添加奖品]                │   │
│  └─────────────────────────────┘   │
│                                     │
│  黑名单: [配置]                     │
│  白名单: [配置]                     │
│                                     │
│           [保存] [重置]             │
└─────────────────────────────────────┘
```

### 4.3 响应式设计
- 桌面端：居中布局，转盘适中
- 移动端：全屏转盘，按钮放大

## 7. 非功能需求

### 7.1 性能要求
- 首次加载时间 < 2秒
- 转盘动画流畅（60fps）
- 本地存储读写即时响应

### 7.2 用户体验
- 操作简单直观
- 配置实时保存
- 错误提示友好
- 支持键盘操作

### 7.3 安全性
- 本地存储数据不包含敏感信息
- 配置数据可随时清除
